## パーソナライズ学習（MVPスコープ）

本アプリのMVPにおけるパーソナライズ学習は、**初回レベル判定テスト**と**レベル別シナリオ表示**にフォーカスする。

### 機能概要

- アプリ初回起動時（ユーザ登録完了直後）に、ユーザーの現在の英語レベルを判定するためのテストを実施する。
- 初回テストの結果に基づき、ユーザーを「初級」「中級」「上級」のいずれかのレベルに自動で振り分ける。
- 振り分けられたレベルに応じて、シナリオ選択画面には対応レベルのシナリオのみが表示される。

### 初回レベル判定テスト

- テストは以下2種類のセクションで構成される。
  - リスニングテスト
  - スピーキングテスト
- 全体で20問を出題する。
  - 問題内容は既存の学習シナリオ（各プロンプト）をベースに作成する。
  - シナリオカテゴリ（旅行・ビジネス・日常会話）からバランスよく出題することを目安とする。
- 各問題には得点を設定し、ユーザーの回答に応じてスコアを加算する。
- 当面は、**リスニング・スピーキング問わず、各問題の得点を単純に合計したスコア**のみをレベル判定に使用する。

### レベル振り分け

- テストの合計スコアに基づき、ユーザーを以下のいずれかに分類する。
  - 初級
  - 中級
  - 上級
- レベル振り分けロジックは、MVP段階では「合計点ベースのシンプルな閾値のみ」とする。
  - 具体的な点数閾値は後続タスクで詳細設計・調整する前提。
  - 将来的には正答率やセクションごとの結果など、より細かい指標を用いることを想定する。

### レベル別シナリオ表示

- シナリオには、それぞれ「初級」「中級」「上級」のレベル属性を持たせる。
- シナリオ選択画面では、ユーザーの現在レベルに一致するシナリオのみを一覧表示する。
  - 例: 初級ユーザーには初級シナリオのみ表示。
- （将来拡張）ユーザーが一定条件を満たした場合にレベルを自動的に更新し、表示されるシナリオを切り替える。

### スコープ外（将来拡張）

以下の機能は、今回のMVPではスコープ外とし、将来拡張として検討する。

- 苦手分野の特定と重点学習（文法・語彙・発音など分野別の弱点分析）
- 学習進捗の可視化（ダッシュボード、学習ストリーク、改善度グラフなど）
- 学習履歴に基づく高度なシナリオ推奨ロジック

なお、上記の詳細設計・実装方針については GitHub Issue #26「パーソナライズ学習の要件・機能設計」を参照する。
